ClickHouse provides extensive array functions for working with array data types, including creation, manipulation, transformation, and aggregation operations.

## Array Creation

### array

Creates an array from arguments.

```sql
SELECT array(1, 2, 3, 4, 5);
-- Result: [1,2,3,4,5]

SELECT array('a', 'b', 'c');
-- Result: ['a','b','c']

-- Shorthand notation
SELECT [1, 2, 3];
-- Result: [1,2,3]
```

### range

Creates an array of numbers in a range.

```sql
SELECT range(5);
-- Result: [0,1,2,3,4]

SELECT range(2, 7);
-- Result: [2,3,4,5,6]

SELECT range(0, 10, 2);
-- Result: [0,2,4,6,8]
```

### emptyArrayUInt8, emptyArrayString

Creates empty arrays of specific types.

```sql
SELECT emptyArrayUInt8();
-- Result: []

SELECT emptyArrayString();
-- Result: []
```

## Array Access

### arrayElement

Accesses an element by index (1-based).

```sql
SELECT arrayElement([1, 2, 3, 4, 5], 3);
-- Result: 3

-- Shorthand notation
SELECT [10, 20, 30, 40][2];
-- Result: 20

-- Negative indices count from end
SELECT [1, 2, 3, 4, 5][-1];
-- Result: 5
```

### arraySlice

Extracts a slice of an array.

```sql
SELECT arraySlice([1, 2, 3, 4, 5, 6], 2, 3);
-- Result: [2,3,4]

SELECT arraySlice([1, 2, 3, 4, 5, 6], -3, 2);
-- Result: [4,5]

-- From test file: 00498_array_functions_concat_slice_push_pop.sql
SELECT arraySlice([1, 2, 3, 4, 5, 6], 2, -2);
-- Slice with negative length
```

## Array Modification

### arrayConcat

Concatenates multiple arrays.

```sql
SELECT arrayConcat([1, 2], [3, 4]);
-- Result: [1,2,3,4]

SELECT arrayConcat(['abc'], ['def', 'gh', 'qwe']);
-- Result: ['abc','def','gh','qwe']

-- From test: 00498_array_functions_concat_slice_push_pop.sql
SELECT arrayConcat([1, NULL, 2], [3, NULL, 4]);
-- Result: [1,NULL,2,3,NULL,4]
```

### arrayPushBack

Adds an element to the end of an array.

```sql
SELECT arrayPushBack([1, 2, 3], 4);
-- Result: [1,2,3,4]

SELECT arrayPushBack(['a', 'b'], 'cd');
-- Result: ['a','b','cd']
```

### arrayPushFront

Adds an element to the beginning of an array.

```sql
SELECT arrayPushFront([1, 2, 3], 0);
-- Result: [0,1,2,3]

SELECT arrayPushFront(['a', 'b'], 'cd');
-- Result: ['cd','a','b']
```

### arrayPopBack

Removes the last element from an array.

```sql
SELECT arrayPopBack([1, 2, 3]);
-- Result: [1,2]

SELECT arrayPopBack(['a', 'b', 'c']);
-- Result: ['a','b']
```

### arrayPopFront

Removes the first element from an array.

```sql
SELECT arrayPopFront([1, 2, 3]);
-- Result: [2,3]

SELECT arrayPopFront(['a', 'b', 'c']);
-- Result: ['b','c']
```

### arrayResize

Resizes an array to a specified length.

```sql
SELECT arrayResize([1, 2, 3], 5);
-- Result: [1,2,3,0,0]

SELECT arrayResize([1, 2, 3, 4, 5], 3);
-- Result: [1,2,3]

SELECT arrayResize([1, 2, 3], 5, 42);
-- Result: [1,2,3,42,42] (fill with 42)
```

## Array Searching

### has

Checks if an array contains an element.

```sql
SELECT has([1, 2, 3, 4], 3);
-- Result: 1

SELECT has([1, 2, 3, 4], 5);
-- Result: 0

SELECT has(['a', 'b', 'c'], 'b');
-- Result: 1
```

### hasAll

Checks if first array contains all elements from second array.

```sql
SELECT hasAll([1, 2, 3, 4], [2, 4]);
-- Result: 1

SELECT hasAll([1, 2, 3, 4], [2, 5]);
-- Result: 0
```

### hasAny

Checks if arrays have any common elements.

```sql
SELECT hasAny([1, 2, 3], [3, 4, 5]);
-- Result: 1

SELECT hasAny([1, 2, 3], [4, 5, 6]);
-- Result: 0
```

### hasSubstr

Checks if array contains a contiguous subsequence.

```sql
SELECT hasSubstr([1, 2, 3, 4], [2, 3]);
-- Result: 1

SELECT hasSubstr([1, 2, 3, 4], [2, 4]);
-- Result: 0 (not contiguous)
```

### indexOf

Returns the index of the first occurrence of an element.

```sql
SELECT indexOf([1, 2, 3, 2, 1], 2);
-- Result: 2

SELECT indexOf([1, 2, 3], 5);
-- Result: 0 (not found)
```

### countEqual

Counts occurrences of an element in an array.

```sql
SELECT countEqual([1, 2, 3, 2, 1], 2);
-- Result: 2

SELECT countEqual(['a', 'b', 'a', 'c', 'a'], 'a');
-- Result: 3
```

## Array Transformation

### arrayMap

Applies a lambda function to each element.

```sql
SELECT arrayMap(x -> x * 2, [1, 2, 3, 4]);
-- Result: [2,4,6,8]

SELECT arrayMap(x -> toString(x), [1, 2, 3]);
-- Result: ['1','2','3']

SELECT arrayMap((x, y) -> x + y, [1, 2, 3], [4, 5, 6]);
-- Result: [5,7,9]
```

### arrayFilter

Filters array elements based on a condition.

```sql
SELECT arrayFilter(x -> x > 2, [1, 2, 3, 4, 5]);
-- Result: [3,4,5]

SELECT arrayFilter(x -> x % 2 = 0, [1, 2, 3, 4, 5, 6]);
-- Result: [2,4,6]
```

### arrayFill

Replaces elements with cumulative sums based on conditions.

```sql
SELECT arrayFill(x -> x > 0, [1, 0, 0, 2, 0, 3]);
-- Fills zeros after positive numbers
```

### arrayReverse

Reverses the order of array elements.

```sql
SELECT arrayReverse([1, 2, 3, 4, 5]);
-- Result: [5,4,3,2,1]

SELECT arrayReverse(['a', 'b', 'c']);
-- Result: ['c','b','a']
```

### arrayFlatten

Flattens nested arrays.

```sql
SELECT arrayFlatten([[1, 2], [3, 4], [5]]);
-- Result: [1,2,3,4,5]

SELECT arrayFlatten([[[1, 2]], [[3, 4]]]);
-- Result: [[1,2],[3,4]] (flattens one level)
```

### arrayCompact

Removes consecutive duplicate elements.

```sql
SELECT arrayCompact([1, 1, 2, 2, 3, 1, 1]);
-- Result: [1,2,3,1]
```

### arrayDistinct

Removes all duplicate elements.

```sql
SELECT arrayDistinct([1, 2, 3, 2, 1]);
-- Result: [1,2,3]
```

## Array Set Operations

### arrayIntersect

Returns elements present in all arrays.

```sql
SELECT arrayIntersect([1, 2, 3], [2, 3, 4]);
-- Result: [2,3]

SELECT arrayIntersect([1, 2], [2, 3], [2, 4]);
-- Result: [2]
```

### arrayExcept

Returns elements from the first array not in the second.

```sql
SELECT arrayExcept([1, 2, 3, 4], [2, 4]);
-- Result: [1,3]
```

## Array Aggregation

### arraySum

Calculates the sum of array elements.

```sql
SELECT arraySum([1, 2, 3, 4, 5]);
-- Result: 15

SELECT arraySum([1.5, 2.5, 3.0]);
-- Result: 7.0
```

### arrayAvg

Calculates the average of array elements.

```sql
SELECT arrayAvg([1, 2, 3, 4, 5]);
-- Result: 3.0
```

### arrayMin / arrayMax

Finds minimum or maximum element.

```sql
SELECT arrayMin([3, 1, 4, 1, 5, 9]);
-- Result: 1

SELECT arrayMax([3, 1, 4, 1, 5, 9]);
-- Result: 9
```

### arrayProduct

Calculates the product of array elements.

```sql
-- From test: 01768_array_product.sql
SELECT arrayProduct([1, 2, 3, 4, 5, 6]) as a, toTypeName(a);
-- Result: 720, Int64

SELECT arrayProduct(array(1.0, 2.0, 3.0, 4.0));
-- Result: 24.0
```

### arrayCumSum

Calculates cumulative sum.

```sql
SELECT arrayCumSum([1, 2, 3, 4, 5]);
-- Result: [1,3,6,10,15]
```

### arrayCumSumNonNegative

Cumulative sum, resetting on negative values.

```sql
SELECT arrayCumSumNonNegative([1, 2, -5, 3, 4]);
-- Result: [1,3,0,3,7]
```

## Array Sorting

### arraySort

Sorts array elements in ascending order.

```sql
SELECT arraySort([3, 1, 4, 1, 5, 9, 2, 6]);
-- Result: [1,1,2,3,4,5,6,9]

SELECT arraySort(['c', 'a', 'b']);
-- Result: ['a','b','c']
```

### arrayReverseSort

Sorts array elements in descending order.

```sql
SELECT arrayReverseSort([3, 1, 4, 1, 5, 9]);
-- Result: [9,5,4,3,1,1]
```

### arrayPartialSort

Partially sorts first N elements.

```sql
SELECT arrayPartialSort(3, [5, 2, 8, 1, 9, 3]);
-- Result: [1,2,3,5,8,9] (first 3 are smallest)
```

## Array Predicates

### arrayAll

Checks if all elements satisfy a condition.

```sql
SELECT arrayAll(x -> x > 0, [1, 2, 3, 4]);
-- Result: 1

SELECT arrayAll(x -> x > 2, [1, 2, 3, 4]);
-- Result: 0
```

### arrayExists

Checks if any element satisfies a condition.

```sql
SELECT arrayExists(x -> x > 3, [1, 2, 3, 4]);
-- Result: 1

SELECT arrayExists(x -> x > 10, [1, 2, 3, 4]);
-- Result: 0
```

### arrayCount

Counts elements satisfying a condition.

```sql
SELECT arrayCount(x -> x > 2, [1, 2, 3, 4, 5]);
-- Result: 3

SELECT arrayCount(x -> x % 2 = 0, [1, 2, 3, 4, 5, 6]);
-- Result: 3
```

## Array Information

### length / arrayLength

Returns the number of elements in an array.

```sql
SELECT length([1, 2, 3, 4, 5]);
-- Result: 5

SELECT arrayLength(['a', 'b', 'c']);
-- Result: 3
```

### empty / arrayEmpty

Checks if array is empty.

```sql
SELECT empty([]);
-- Result: 1

SELECT empty([1, 2, 3]);
-- Result: 0
```

### notEmpty / arrayNotEmpty

Checks if array is not empty.

```sql
SELECT notEmpty([1, 2, 3]);
-- Result: 1
```

## Advanced Array Functions

### arrayEnumerate

Returns array of element indices.

```sql
SELECT arrayEnumerate([10, 20, 30]);
-- Result: [1,2,3]
```

### arrayDifference

Calculates differences between adjacent elements.

```sql
SELECT arrayDifference([1, 3, 6, 10, 15]);
-- Result: [0,2,3,4,5]
```

### arrayReduce

Applies an aggregate function to array elements.

```sql
SELECT arrayReduce('sum', [1, 2, 3, 4, 5]);
-- Result: 15

SELECT arrayReduce('avg', [1, 2, 3, 4, 5]);
-- Result: 3.0
```

## String-Specific Array Functions

### arrayStringConcat

Joins array of strings with a separator.

```sql
SELECT arrayStringConcat(['Hello', 'World'], ' ');
-- Result: 'Hello World'

SELECT arrayStringConcat(['a', 'b', 'c'], ',');
-- Result: 'a,b,c'
```

## Performance Considerations

1. **Array operations are vectorized** for high performance
2. **Use `has` instead of `IN`** for array membership checks
3. **Lambda functions** in `arrayMap` and `arrayFilter` are optimized
4. **Nested arrays** have performance overhead
5. **Large arrays** should be used judiciously in GROUP BY

## Common Patterns

```sql
-- Filter and transform in one go
SELECT arrayMap(x -> x * 2, arrayFilter(x -> x > 0, [-1, 2, -3, 4]));
-- Result: [4,8]

-- Remove nulls from array
SELECT arrayFilter(x -> isNotNull(x), [1, NULL, 2, NULL, 3]);
-- Result: [1,2,3]

-- Check if any element matches
SELECT arrayExists(x -> x LIKE '%error%', log_messages) FROM logs;
```
